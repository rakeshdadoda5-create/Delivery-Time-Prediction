# 🚚 Delivery Time Prediction System for Quick Commerce

## 📌 Project Overview

This project predicts delivery time for quick commerce orders using Machine Learning (LightGBM). It analyzes operational factors such as traffic, weather, rider experience, and store preparation time to estimate delivery time accurately.

## 🎯 Business Problem

Quick commerce companies need accurate ETA predictions to:
- Improve customer satisfaction
- Optimize rider allocation
- Reduce delivery delays
- Improve operational efficiency

## 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- LightGBM
- Streamlit

## 📂 Dataset

The project uses three datasets:
- orders.csv
- riders.csv
- stores.csv

These datasets are merged to build the final training dataset.

## ⚙️ Features Used

- Distance
- Traffic
- Weather
- Batch Order
- Rider Experience
- Vehicle Type
- Rider Rating
- Store Preparation Time

## 🤖 Machine Learning Model

- LightGBM Regressor

## 📈 Evaluation Metrics

- MAE
- RMSE
- R² Score

## 💻 Streamlit Dashboard

The application allows users to:
- Enter delivery details
- Predict delivery time
- View estimated ETA instantly
